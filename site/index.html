Usage Guide

1. Include via CDN

Add these to your HTML <head>:

<link rel="stylesheet" href="https://EnesKeremAYDIN.github.io/palette.css/palette/palette.css">
<script src="https://EnesKeremAYDIN.github.io/palette.css/palette/theme-loader.js"></script>

2. Set Theme Attributes

Add data-* attributes to your <body>:

<body 
  data-theme="forest"
  data-radius="soft"
  data-font="serif"
  data-shadow="deep"
  data-spacing="relaxed"
  data-anim="smooth"
>

3. Write HTML

Write your HTML normally – styles are automatically applied:

<h1>Your Heading</h1>
<p>Your paragraph text.</p>
<button>Click me</button>
<input type="text" placeholder="Enter text">

4. Advanced: Custom Themes

Create your own theme by adding a CSS file to palette/palettes/colors/ and set data-theme to your theme name.

/* palette/palettes/colors/mybrand.css */
:root {
  --bg: #f8f9fa;
  --text: #212529;
  --primary: #6f42c1;
  --primary-hover: #5a32a3;
  --on-primary: #fff;
}
<body data-theme="mybrand">
